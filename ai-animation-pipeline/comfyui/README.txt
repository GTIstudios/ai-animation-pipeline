# ComfyUI Workflows

This folder contains the core ComfyUI JSON graphs used in GTI Studios’ AI animation pipeline.

## Included Workflows

### `01_reference_setup.json`
Base reference → style consistency loader.

### `02_character_stylelock.json`
Locks character style for Space Bear, Onyx, and other character models.

### `03_action_scene_animation.json`
AnimateDiff + WAN22 setup for action sequences (e.g., reactor room, docking scenes).

### `04_lip_sync_setup.json`
Prepares frames for lip-sync workflows and timing alignment.

### `05_video_patch_flow.json`
Video-to-video patching, frame consistency, and detail enhancement.
